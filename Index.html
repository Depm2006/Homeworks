Esta pagina web fue creada como practica del modulo FSMOL3|HTML que forma parte del contenido del curso.

Los componentes de dicho modulo son:

  1. Concepto de HTML
  2. Etiquetas de Formato
  3. Etiquetas de Bloques
  4. Etiquetas de Contenido

Cada uno de estos componentes esta formado, como su nombre lo indica, por etiquetas y argumentos propios de cada una.

    html:Raiz de un documento HTML, engloba todo el contenido de la pagina.

    Etiquetas de Formato: Ayudan a dar forma e identificar los diferentes titulos y secciones que conforman una pagina web.

      Etiquetas de Bloques: Ayudan a clasificar la informacion presentada por bloques de manera ordenada dentro del codigo

        Header
        Main
        Footer

      Etiquetas de contenido:

        IMG
        Form
        Input (type/name/etc)
        Label
